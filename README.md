## Visão Geral

Nordic é um site de portfólio para agência posicionada no mercado de IA. O projeto foca em excelência visual: cada componente utiliza animações GSAP com SplitText, SVGs com stroke animado via `pathLength`, efeito de fluido WebGL iridescente como overlay global, e smooth scroll nativo com Lenis.

## Componentes

### `SvgHoverCard`

O componente central do projeto. Exibe um card com imagem ou vídeo de fundo, e ao hover revela:
- Stroke path SVG animado via `strokeDashoffset` (GSAP)
- Logo do cliente com fade-in
- Título animado com `SplitText` (word by word)
- Headline e descrição com `SplitText` (char by char)
- Tags no rodapé

```tsx
import { SvgHoverCard } from '@/components/svg-hover-card/svg-hover-card'

<SvgHoverCard
  id="card-1"
  imageSrc="/img.jpg"           // ou videoSrc="/video.mp4"
  strokeColor="#64BEF6"
  strokeColorSecondary="#EAEBEC"
  logoSrc="/logo.svg"
  logoAlt="Client"
  externalUrl="https://client.com"
  headline="+5K/mo"
  description="Description of the work done for this client."
  tags={['Webflow', 'Branding']}
/>
```

### `LogoMark`

Marquee infinito de logos de parceiros animado com GSAP. Anima o track com `x: 0 → -listWidth` e `repeat: -1` para loop contínuo. Inclui gradientes de fade nas bordas.

```tsx
import { LogoMark } from '@/components/logo-mark'

<LogoMark
  logos={[
    { src: '/webflow.svg', alt: 'Webflow' },
    { src: '/rubi.svg', alt: 'Ruby' },
  ]}
  speed={50}
  fadeColor="white"
/>
```

### `Fluid`

Overlay WebGL fixo sobre toda a página com efeito de fluido interativo (reage ao mouse). Renderiza transparente sobre o conteúdo via `zIndex: 9999` e `pointerEvents: none`.

```tsx
// Já incluso no layout.tsx
import { Fluid } from '@/components/fluid'

<Fluid />
```

**Efeitos inclusos:** Bloom, Noise, Vignette e shader iridescente custom (Fresnel + chromatic aberration + hue shift por gradiente de luminância).

### `HeroSection`

Navbar + headline principal. Copy: *"AI companies deserve more than agencies"*.

### `ConnectorsSection`

Seção com cena 3D de conectores usando `@react-three/fiber` e `@react-three/rapier` para física.

### `ServicesSection`

Lista de serviços com CTA de contato.

### `FooterSection`

Rodapé com depoimentos de clientes (testimonials).

### `ScrollToExplore`

Indicador de scroll animado.

### `TwinArrow`

Componente de seta dupla animada (entrada/saída) com GSAP, com suporte a hover no elemento pai via `hoverParent`.

### `ScrollBar`

Barra de progresso de scroll customizada.

## Instalação

### Pré-requisitos

- Node.js 18+
- pnpm

### Setup

```bash
# 1. Clone o repositório
git clone https://github.com/eryckassis/svg-hover-card-component.git
cd svg-hover-card-component

# 2. Instale as dependências
pnpm install

# 3. Inicie o servidor de desenvolvimento
pnpm dev
```

Acesse [http://localhost:3000](http://localhost:3000).
